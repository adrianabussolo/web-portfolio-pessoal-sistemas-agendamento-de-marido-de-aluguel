Objetivo:
Criar uma aplicação web para agendamento de marido de aluguel.

Contexto:
 - A aplicação web será baseada na API para agendamento de marido de aluguel que está definida no Swagger file armazenado em resources\swagger.yaml
 - Cadastro de usuario e cadastro de serviço e listagem de serviço
 - Autenticação de usuários via JWT
 - Documentação interativa via Swagger UI
 - Armazenamento dos dados em memória
 - A API estará rodando em http://localhost:3000

Regras:
- Apenas execute o que eu te pedir, não me pergunte nada.
- Crie esta aplicação web usando o framework express do Javascript.
- Utilize HTML e CSS para o frontend, use algum framework de estilização de frontend como o Bulma para estilização.
- Faça a aplicação WEB rodar na porta 4000.
- Configure mensagens de erro para quando a API retornar falhaas (fora da faixa de status code 200).
